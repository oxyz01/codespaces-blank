# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-17

### Added
- Initial release of professional Kali Linux WSL 2 installer
- Automated WSL 2 feature installation
- Official Kali Linux distribution installation from Microsoft package
- Comprehensive error handling and recovery
- Detailed logging with timestamps and severity levels
- Pre-flight system validation (Windows version, disk space, admin rights)
- Automatic initialization and package updates
- Optional SSH server installation and configuration
- Optional pentesting tools installation
- Configurable installation parameters
- Intelligent rollback on failure
- Professional README with troubleshooting guide
- Support for both one-liner and local execution
- Clear post-installation guidance

### Features
- ✅ Windows 10 Build 18362+ and Windows 11 support
- ✅ Admin privilege verification
- ✅ Disk space validation
- ✅ WSL 2 automatic setup
- ✅ Kali Linux official package installation
- ✅ Unattended initialization
- ✅ Base tools installation
- ✅ SSH server configuration
- ✅ Pentesting tools installation
- ✅ Distribution configuration
- ✅ Comprehensive logging
- ✅ Installation progress tracking

### Known Limitations
- Requires administrator privileges
- Windows 10 Build 18362+ or Windows 11 only
- Internet connection required
- ~15GB minimum free disk space required
